趣味赚 基于 uniapp +tp6
===============
> 运行环境要求 ThinkPHP 6.0

> 运行环境要求 PHP7.1+

> 运行环境要求 MongoDB 4.0.10


---

支持生成 h5 安卓 ios 微信小程序 支付宝小程序 百度小程序 字节跳动 等

## 安装

~~~
composer create-project topthink/think tp 6.0.*-dev
~~~

如果需要更新框架使用
~~~
composer update topthink/framework
~~~

消息推送
~~~
php think worker:gateway
~~~
